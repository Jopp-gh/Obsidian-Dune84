# CHEATSHEET - CSSCLASSES
```
cssclasses are case sensitive eg. `Banner-a` ≠ `banner-a`
```

<br>

## TABLES

### TABLE ALIGNMENT

`cssclass: table-c`

center tables

`cssclass: table-eq`

equally distribuited columns

`cssclass: table-wide`

stretch tables

`cssclass: table-hl`

headless tables

**Deutsch**
`cssclass: tabelle-m`

m = Mitte

`cssclass: tabelle-gs`

gs = gleiche Spalten(weite)

`cssclass: tabelle-weit`

weit = ganze Breite

`cssclass: tabelle-kl`

kl = kopflos

<br>

### TABLE COLORS

```
cssclass: table-r
cssclass: table-g
cssclass: table-b
cssclass: table-m
```

classic header tint - red / green / blue / monochrome

```
cssclass: table-xyr
cssclass: table-xyg
cssclass: table-xyb
cssclass: table-xym
```

double-header tint - red / green / blue / monochrome

**Deutsch**
```
cssclass: tabelle-r
cssclass: tabelle-g
cssclass: tabelle-b
cssclass: tabelle-m
```
klassische Tabelle

```
cssclass: tabelle-xyr
cssclass: tabelle-xyg
cssclass: tabelle-xyb

```
doppelköpfige Tabelle

<br>

### TABLE HIGHLIGHTS

`cssclass: table-fc`

highlight first column

`cssclass: table-lc`

highlight last column

`cssclass: table-alt`

highlight alternate columns

`cssclass: table-lr`

highlight last row

**Deutsch**

`cssclass: tabelle-es`

es= erste Spalte

`cssclass: tabelle-ls`

ls = letzte Spalte

`cssclass: tabelle-alt`

alt = alternierend

`cssclass: tabelle-lr`

letzte Reihe
