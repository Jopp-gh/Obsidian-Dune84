# CHEATSHEET - CSSCLASSES
cssclasses are case sensitive eg. `Banner-a` ≠ `banner-a`

<br>

## TABLES

### TABLE ALIGNMENT
center tables

`cssclass: table-c`

equally distribuited columns

`cssclass: table-eq`

stretch tables

`cssclass: table-wide`

headless tables

`cssclass: table-hl`

<br>

#### Deutsch
m = Mitte

`cssclass: tabelle-m`

gs = gleiche Spalten(weite)

`cssclass: tabelle-gs`

weit = ganze Breite

`cssclass: tabelle-weit`

kl = kopflos

`cssclass: tabelle-kl`


<br>

### TABLE COLORS
classic header tint - red / green / blue / monochrome

```
cssclass: table-r
cssclass: table-g
cssclass: table-b
cssclass: table-m
```

double-header tint - red / green / blue / monochrome

```
cssclass: table-xyr
cssclass: table-xyg
cssclass: table-xyb
cssclass: table-xym
```

<br>

#### Deutsch
klassische Tabelle

```
cssclass: tabelle-r
cssclass: tabelle-g
cssclass: tabelle-b
cssclass: tabelle-m
```

doppelköpfige Tabelle

```
cssclass: tabelle-xyr
cssclass: tabelle-xyg
cssclass: tabelle-xyb

```

<br>

### TABLE HIGHLIGHTS
highlight first column

`cssclass: table-fc`

highlight last column

`cssclass: table-lc`

highlight alternate columns

`cssclass: table-alt`

highlight last row

`cssclass: table-lr`

<br>

#### Deutsch
es= erste Spalte

`cssclass: tabelle-es`

ls = letzte Spalte

`cssclass: tabelle-ls`

alt = alternierend

`cssclass: tabelle-alt`

letzte Reihe

`cssclass: tabelle-lr`

## IMAGES

### BANNER
top
`cssclass: banner-a` and `![[image.png|banner-a]]`
bottom
`cssclass: banner-b` and `![[image.png|banner-b]]`
everywhere 
`![[image.png|banner-m]]`

### Simple image descriptions
cssclass: `capt-10`, `capt-15`, `capt-20`, `capt-25`, `capt-30`, `capt-35`, `capt-40`

### Simple Image grid 

Cssclass: `grid-10`, `grid-20`, `grid-30`, `grid-40`, `grid-15`, `grid-25`, `grid-35`, `grid-45`
