## TABLES

### TABLE ALIGNMENT

`cssclass: table-c`

center tables

`cssclass: table-eq`

equally distribuited columns

`cssclass: table-wide`

stretch tables

`cssclass: table-hl`

headless tables

<br>

### TABLE COLORS


### TABLE HIGHLIGHTS

`cssclass: table-fc`

highlight first column

`cssclass: table-lc`

highlight last column

`cssclass: table-alt`

highlight alternate columns

`cssclass: table-lr`

highlight last row
