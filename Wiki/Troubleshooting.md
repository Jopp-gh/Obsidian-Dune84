## Troubleshooting

### Check your installation
1. Download/update and activate the **Style settings** plugin. *Please don't miss this step*
2. Download/update Dune-Gold, the theme
3. Download/update Dune's companion snippets (Dune's mermaid theme, Dune's folder icons, Dune's supported plugins) *-optional-*

### Dune configuration
Have `Style Settings` installed and activated, then open the command palette with `Ctrl+P` (Win, Linux) or `Cmd+P` (Mac) and type `style settings` to get `style settings: show style settings view` and confirm. Now, Style settings will open as a note. Pick its tab and drag this window to the rightor left side dock of Obsidian. From now on, you can tweak Dune's interface to your hearts content, Nice!

### Unable to access Obsidian Settings (phone)
to open Settings, you can open the command palette with a swipe from the top border down. With the command palette open, type `settings` and confirm the suggestion `Open settings`

### Slow interface
Please check if you have the latest version of Dune.
To make sure you're always up-to-date, open Obsidians `Settings>Appearance>` then click on `Check for updates`

### Interface looks odd
A possible cause is an outdated version of Dune.
Yet another cause are incompatible snippets, trying to change the same rules as Dune modifies. Finally, please check if you need to update Dune's companion snippets, if you use these snippets too.
Obsidian is changing all times and Dune needs to adapt to this develpment process. Sometimes the changes in Obsidian are minor, sometimes major and due to these circumstances, some features of Dune may not work as expected or break completely. 

### Dune hides everything
Open `Dune>Show` to reveal hidden elements in Dune.
Hidden interface elements should help you to focus better on your work. If you are a newcomer, try first to familiarize with Obsidians interface, before you hide or deactivate something.

### Large margins
if you experience large margins, try to deactivate `Settings>Editor>Readable line lenght` as this option is usually on in Obsidian by default.

### Large fonts
Reset font size to defaults in `Dune>Fonts>Typesets` and `1 - Modern` or `2 - Poetry` or `3 - Dashboard` with the slider `Custom Font size`

### Footnotes disappeared
If you use `banner-b` which is a foot banner, remove that banner

### Problems with fonts
open `Dune>Fonts` . In there, delete and re-enter some Character in the input fields to refresh fonts. Alternatively, click the `Restore default` Button at the end of the input field.

### Typesetting cssclass
Typesetting **cssclass** `ftheme-1`, `ftheme-2` and `ftheme-3` applies only to the current note in Preview View

### Callout framed-a
If typesetting alignment is set to justified, then the callout `framed-a` won't align at the center.

### Problems with colors
if Dune is unable to display some color, go to`Dune>Colors`, click on the color you miss and in the popup, press `Save`. Alternatively, click the `Restore default` Button. This should refresh Dune's color palette as well.

### About the accent color
the accent color of Obsidian works only in a limited way in Dune. I may change this behavior in future versions of Dune, if there is interest.

### Other Issues
Check your CSS snippets under `Settings>Appearance>CSS snippets` and deactivate them all to see if this solves your issue.
If you're using my companion snippets, try to download them again, to make sure you have the latest version of these snippets.

### Exit from zoomed view (phone)
To exit from zoomed images, tab on the left of right screen border.

### Note title is visible all times (phone)
Just tab on the title field and then, tab away on your note. This should hide the note title again.

### Top margin in edit mode (phone)
the top bar is a small and thight space, with title, path bar, the (superfluous) left sidebar button and some more buttons in addition, if you install plugins, (example, **Commander plugin** to name one of them) . These interface elements will inevitably, operlap with your notes, making editing much harder at the top page. Additionally, in edit mode OBSIDIAN jumps abruptly to the top, when the keyboard shifts in.
With Dune, the switch between preview mode and edit mode / live preview is less abrupt but more smooth and you can edit your first line always without worrying that something (buttons or names) gets in the way.
